STUDENT ROSTER

This website will prompt you for four commands.they are ADD,REMOVE,DISPLAY,QUIT

  1. ADD -create a prompt for the student name.Adding the name to the student array in javascript file
  2. REMOVE -create a prompt for student name request.Remove the name from roster array
  3. DISPLAY -print the roster array using console.log
  4. QUIT -should end the web page
